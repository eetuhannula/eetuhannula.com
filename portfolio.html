Web Development

This website is served from Apache web-server running on my Linux box, hosted at Linode. It runs on Ubuntu server OS.

I decided to host my website this way, because it required me to build it from the ground up, without relying on website-hosting platform. This way I learned the basics of Linux servers and Apache2. It was also a great way to get introduced into cloud computers as I had no good option to "self host" in any other way.
